# Contributing

Thanks for taking the time to improve TempTimer.

## Local Development

Open `index.html` in a browser or serve the directory with:

```sh
python3 -m http.server 8000
```

## Before Opening a Pull Request

- Keep the app dependency-free unless there is a clear reason to add a dependency.
- Verify the timer flow in a browser.
- Check the mobile layout in a narrow viewport.
- Run:

```sh
node --check script.js
```

## Pull Request Guidelines

- Keep changes focused.
- Describe the user-facing behavior that changed.
- Include screenshots for visual changes when possible.
