# Security Policy

## Supported Versions

TempTimer is a static browser app. The latest version on the default branch is the supported version.

## Reporting a Vulnerability

Please report security issues privately to the repository owner instead of opening a public issue.

Include:

- A short description of the issue.
- Steps to reproduce it.
- The browser and operating system used.
- Any suggested fix, if known.

The project maintainer will review the report and coordinate a fix when needed.
